# Sales Data Analysis & Dashboard

This project analyzes retail sales data using PostgreSQL and Excel.

## Tools Used
- PostgreSQL (SQL Queries, Aggregations)
- Microsoft Excel (Pivot Tables, Charts, Dashboard)

## Key Analysis
- Monthly Sales Trend
- Top Selling Products by Revenue
- Region-wise Sales Performance

## Output
An interactive Excel dashboard for business reporting and decision making.
